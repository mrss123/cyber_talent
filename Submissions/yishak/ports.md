**question**

what the 30 most known ports and how are they attacked 

## port 21 

FTP (file transfer protocol) : bruteforce attackes , anonymous login exploitation , data intecetpion.

## port 22 

SSH  ( Secure Shell): Brute-force credential cracking and exploitation of weak keys.
    
## port 23 

Telnet: Plaintext credential sniffing, session hijacking, and brute-force due to lack of encryption.
    
## port 25 

SMTP (Simple Mail Transfer Protocol): Open relay abuse to send spam, mail spoofing.
    
## port 53 

DNS (Domain Name System): DNS amplification/reflection DDoS attacks, cache poisoning.
    
## port 69 

TFTP (Trivial File Transfer Protocol): Unauthorized file retrieval and directory traversal.
    
## port 80 

HTTP (Hypertext Transfer Protocol): Web application attacks (SQL injection, XSS), DDoS.
    
## port 110 

POP3 ( Post Office Protocol): Brute-force credentials, data interception.
    
## port 123 

NTP (Network Time Protocol): Reflection amplification DDoS attacks.
    
## port 135 

MS RPC ( Remote Procedure Call): DCOM exploitation, lateral movement.
    
## port 137/139 

NetBIOS: Exploiting misconfigured Windows shares, null sessions.
    
## port 143 

IMAP (Internet Message Access Protocol): Brute-force, clear-text credential sniffing.
    
## port 161/162 

SNMP ( Simple Network Management Protocol): Community string guessing, information disclosure.
    
## port 389 

LDAP (Lightweight Directory Access Protocol): Anonymous binding, injection attacks.
    
## port 443 

(HTTPS - Secure Web Protocol): SSL/TLS misconfiguration, web application exploits.
    
## port 445 

SMB (Server Message Block): EternalBlue (MS17-010), ransomware, SMB relay attacks.
    
## port 465 

(SMTPS): Brute-force, SSL misconfiguration.
    
33 port 514 

(Syslog): Log injection, information leakage.
    
## port 515 

LPD (Line Printer Daemon): Print job hijacking, DoS attacks.
    
## port 995 

(POP3S): Brute-force, SSL misconfiguration.
    
## port 1433/1434 

(MSSQL - Microsoft SQL Server): SQL injection, brute-force, database exploitation.
    
## port 1521 

(Oracle DB): TNS listener poisoning, credential brute-force.
    
## port 1723 

(PPTP - VPN): Brute-force, connection hijacking.
    
## port 1900 

UPnP (Universal Plug and Play): Reflection amplification DDoS.
    
## port 3306 

(MySQL): Unauthorized access, database hacking.
    
## port 3389 

RDP (Remote Desktop Protocol): BlueKeep (CVE-2019-0708), brute-force password attacks.
    
## port 5432 

(PostgreSQL): Default credential exploitation, SQL injection.
    
## port 5900 

VNC (Virtual Network Computing): No authentication access, brute-force.
    
## port 6379 

(Redis): Unauthenticated access, Remote Code Execution (RCE).
    
## port 8080 

(HTTP Alternate): Web proxy misconfiguration, admin panel exploitation.
